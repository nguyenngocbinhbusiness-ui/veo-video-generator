# Business Analysis Versions

## Overview
This directory contains versioned business analysis documents for the project.

## Versions Index

| Version | Date | Status | Change Summary |
|---------|------|--------|----------------|
| [v1](./business_analysis_v1.md) | 2025-12-21 | approved | Initial version - AI Chat Feature |

## Naming Convention
- Format: `business_analysis_v{N}.md`
- Example: `business_analysis_v1.md`, `business_analysis_v2.md`

## When to Create New Version
- Major feature additions
- Significant requirement changes
- Project scope changes
- At start of new sprint/iteration

## Version Status
- `draft` - Work in progress
- `review` - Pending review
- `approved` - Approved for use
- `deprecated` - Superseded by newer version

## Metadata Fields
Each version file includes YAML frontmatter with:
- `document_type`: Type of document
- `version`: Version number
- `version_date`: Date created
- `author`: Creator
- `status`: Current status
- `change_summary`: Description of changes
- `related_documents`: Links to companion documents
